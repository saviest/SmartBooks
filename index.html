Vision

SmartBooks is an intelligent system that allows students and professors to find the magazines and books easier and quicker than the traditional way of searching material at the library. This system pretends to help the users on their search of a material from the library. The principal way of helping them is to have a personalized search based an account which contains selections, enrolled courses and past rated books and magazines taken from the library. The user will also have an easy way to find the material in the library because of an interactive system of lights that will show the location of the material needed just by selecting the material needed. The system is pretend to work by an android/apple/computer application with a customizable environment that will be connected to the ambient of the library via Wi-Fi sending the notifications and signals to the system so it can work correctly and work with the right process for the right user. The project is meant to be used principally by students and professors but it will also help dependants of the library, workers of the Politecnico di Torino, book providers and cleaners attendants.

AmI features

AmI feature Description Sensitive Trougth the NFC sensors the app can turn on . Responsive Through the led Lights insides the library, the user can notice the location of book, that they need. Adaptive The system  can detect who is using the system, and could recommend which kinds of books could be attractive to the user, showing in parallel of the search of the user. Transparent Due to the little sensors and the platform in the cellphone the system could look like insivible. Ubiquitous Since the user's entrance in the library the system follows the user a long his stay. Intelligent Trough of the database, the system could be able to know the preferences of the user and recommend new books or material.
